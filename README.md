# lcd-driver-lib

Drivers for LCD and OLED displays.

The drivers are organized by programming language. Each subfolder contains a README file describing which display and which processor the driver targets.

## Supported Displays

- Pico OLED 1.3"
- Pico LCD 1.14"
- FireBeetle Covers OLED 128x64 Display
- LCD 16x2 (Epson-style controller)
